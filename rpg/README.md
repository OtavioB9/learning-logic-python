# RPG Character Exercise

Pequeno exercício em Python para criar e validar um personagem de RPG.

## O que o código faz

- Valida o nome do personagem.
- Valida os atributos `strength`, `intelligence` e `charisma`.
- Retorna mensagens de erro quando os dados não seguem as regras.
- Retorna o personagem formatado quando tudo está correto.

## Regras

- O nome deve ser uma string.
- O nome não pode estar vazio.
- O nome não pode ter mais de 10 caracteres.
- O nome não pode conter espaços.
- Todos os atributos devem ser inteiros.
- Todos os atributos devem estar entre 1 e 4.
- A soma dos atributos deve ser 7.
